## 환경 설정

```
$ npm install -g nightwatch selenium-standalone
$ selenium-standalone install --drivers.chrome.version=2.20 --drivers.chrome.baseURL=https://chromedriver.storage.googleapis.com
$ selenium-standalone start
```
